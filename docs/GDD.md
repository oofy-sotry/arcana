## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 플랫폼 | 데스크톱 (Electron) |
| 핵심 재미 | 육성 (메인) / 전투 (보조) / 수집 (필수) |
| 배경 세계관 | 디지몬 + 포켓몬 + 유희왕 혼합 |
| 플레이어 역할 | 계약자 (에레멘탈과 계약하는 인간) |
| 특징 | 펫이 실제 데스크톱 화면 위를 돌아다님 |

### 핵심 차별점
- 펫이 단순 창 안의 존재가 아니라 **실제 바탕화면 위를 돌아다님**
- **2가지 성장 축** (나이 + 레벨)으로 플레이 방식 다양화
- **속성 기반 수집** — 다양한 속성과 진화 경로
- **교배 체인**으로 극희귀 전속성 캐릭터 생성 가능
- **방치해도 살지만, 관심 주면 압도적으로 강해지는** 설계

## 2. 핵심 루프

```
[데일리 루프]
  접속 → 컨디션 확인 → 돌봄 (밥/씻기/놀기)
       → 사냥터 / 보물탐사 → 퀘스트 → 결과 확인
       → 스킬 강화 / 장비 조정 → 교배 / 온라인 활동

[장기 루프]
  레벨업 → 스킬 강화 → 진화 달성
        → 히든 진화 조건 추적 → 교배 전략 수립
        → 혼합 속성 체인 → 옴니렉스 도전

[수집 루프]
  코인 가챠 / 사냥 드롭 / 교배
        → 새 속성 에레멘탈 획득
        → 파티 조합 → 시너지 최적화
```

### 능동 vs 방치 설계 원칙
```
방치: 컨디션 천천히 감소, 자동 사냥 (에너지 소진까지), 기본 성장
능동: 수동 사냥 (에너지 절반 소모, 드롭률 +20%), 돌봄 직접 수행
      (친화도 +50%), 스킬 타이밍 직접 발동 (+30% 효과)

핵심: 방치해도 생존, 능동 플레이 시 압도적으로 빠른 성장
목표 접속 주기: 하루 1~2회로 펫 생존 유지 가능
```

## 3. 세계관 및 스토리

### 세계관 설정
```
아르카나 (Arcana)
  현실 세계와 평행한 디지털/영적 세계
  속성 생명체 "에레멘탈"이 거주
  플레이어 = 계약자 — 에레멘탈과 계약해 현실에 소환

아르카 (Arca)
  태초의 근원 에너지, 의지를 가진 시원적 존재
  권태를 느껴 스스로를 빛과 어둠으로 분열시킴
  두 극의 영원한 갈등 에너지를 섭취하며 자기 증식
  균열 = 아르카의 섭식 기관 (빨대)

주요 세력
  Luxis (빛의 군세)
  Noctis (어둠의 무리)
  계약자 (플레이어 — 어느 세력에도 속하지 않는 변수)
```

### 스토리 구조 (5챕터 + 히든 엔딩)

**1장 — 첫 계약 (각성)**
- 균열 발생, 첫 에레멘탈 **이그니스(불)** 와 계약 (튜토리얼 파트너)
- **영혼 동화도 게이지** 첫 등장 (UI에 "????" 표시, 복선)
- 균열에서 나오는 폭주 에레멘탈: 빛도 어둠도 아닌 공허한 눈빛 *(복선)*

**2장 — 여덟 속성 지역 탐방**
- 8개 지역(8가지 일반 속성) 탐방, 동료 에레멘탈 영입
- 각 지역마다 도덕적 선택 → 진영 평판 + 동료 충성도 변동
- 어떤 선택을 해도 균열은 심화 → 위화감 누적
- 각 지역 장로의 공통 대사: *"빛과 어둠은 너무 오래, 너무 완벽하게 싸워왔다."*

**3장 — 혼합 속성 등장 + 대반전**
- 혼합 에레멘탈 자연 발생, 사회는 배척
- 발견: **혼합 존재만이 아르카의 섭식에 면역**
- 진실 폭로: 빛/어둠 지도자들이 아르카에게 조종당하고 있음
- 영혼 동화도 정체 공개: "주인공이 아르카가 소화할 수 없는 혼합 존재가 되어가는 중"
- 빛/어둠 속성 해금

**4장 — 거짓된 전면전**
- Luxis vs Noctis 전면 충돌, 진짜 승자는 아르카
- 목표: 전쟁을 멈추는 것 (이기는 것이 아님)
- 2~3장 누적 선택 결과 반영
  - 충성도 70+: 동료가 진실을 믿고 따름
  - 충성도 30~70: 설득 필요
  - 충성도 30-: 아르카 영향에 휩쓸려 배신

**5장 — 옴니렉스 (정규 엔딩)**
- 10속성 모두와 계약 → 전속성 시원체 **옴니렉스(Omnirex)** 소환
- 아르카를 잠재우고, 섭식 기관(균열) 제거
- 두 세계 안정적 경계 회복, 아르카는 잠들 뿐 (영구 봉인 아님)

**히든 엔딩 — 카오스(混)**
- 해금 조건:
  1. 혼합 존재 전부 구원 (3장 도덕 선택)
  2. 진영 평판 균형 유지 (Luxis-Noctis 차이 ±20 이내)
  3. 영혼 동화도 최대치 도달
- 옴니렉스로 봉인 대신 → **아르카를 흡수**
- 주인공이 새로운 시원 존재 **카오스(Chaos)** 로 변모
- 빛/어둠 재융합, 현실과 아르카나 하나가 됨
- "혼돈=악"의 완전한 해체

### 신규 시스템 (스토리 연동)
| 시스템 | 설명 |
|--------|------|
| 영혼 동화도 | 계약/친화도 누적 시 상승, 처음엔 숨김, 히든 엔딩 조건 |
| 동료 충성도 | 지역 선택으로 변동, 4장 행동 결정 |
| 진영 평판 | Luxis / Noctis 양방향 수치 |
| 혼합 구원 기록 | 3장 선택 누적, 히든 엔딩 조건 |

## 4. 시간 시스템

```
12 실시간 시간 = 1 게임 나이 (1살)

나이별 기준:
  50살  = 실시간 25일  (자연사 확률 시작)
  100살 = 실시간 50일
  140살 = 실시간 70일  (극한 히든 진화 조건)

오프라인 처리:
  시간 흐름: 오프라인 중에도 나이 증가
  컨디션:   -1.5% / 실시간 시간 자동 감소
  오프라인 안전 접속 주기: 약 이틀에 한 번
```

### 전속성 — 옴니렉스 (Omnirex)
```
모든 속성의 힘을 하나에 담은 유일한 존재
생성 방법: 혼합 속성 교배 체인 (1% 확률 누적)
  예) 물+불(증기) × 바람 → 1% → 물+불+바람
      물+불+바람 × 땅 → 1% → 물+불+바람+땅

총 확률: 약 0.01^9 (사실상 불가능에 가깝지만 존재)
특성: 모든 속성 상성 무효, 교배 불가
```

## 6. 캐릭터 (에레멘탈) 시스템

### 진화 단계
```
유년기 → 성장기 → 완전체 → 궁극체 → 전설체
```

### 캐릭터 등급
| 등급 | 설명 | 획득 방법 |
|------|------|-----------|
| 일반 | 8속성 기본 라인 | 가챠, 사냥 드롭, 교배 |
| 히든 | 각 속성별 히든 진화체 | 숨겨진 조건 충족 |
| 히든 속성 | 빛, 어둠 | 3장 스토리 클리어 |
| 혼합 속성 | 불호환 교배 성공 | 극저 확률 교배 |
| 전속성 | 옴니렉스 | 혼합 속성 교배 체인 |

### 규모 (MVP 기준)
```
일반 8속성 × (5 기본 + 4 히든) = 72 캐릭터
히든 2속성 × (5 기본 + 4 히든) = 18 캐릭터
혼합 속성 14종                  = 70+ 캐릭터 (5단계)
옴니렉스                        = 1 캐릭터
총계                            = 161+ 캐릭터 (최종 목표)
MVP 최소                        = 50 캐릭터 (기본 라인만)
```

### 캐릭터 초안 (기본 라인)

## 7. 성장 시스템

### 스탯 성장 공식
```
스탯 증가 = 나이 기여(30%) + 레벨 기여(60%) + 상황 보너스(10%)

나이 기여:   실시간이 흐를수록 자동 증가
레벨 기여:   사냥/퀘스트 등 활동으로 경험치 획득
상황 보너스: 친화도 + 컨디션 → 렙업 시 추가 스탯 획득 확률
             (친화도/컨디션이 좋을수록 확률 상승)
```

### 친화도 / 신뢰도
```
친화도 (Affinity) — 펫 ↔ 주인공
  범위: -100 ~ 100
  영향: 진화 조건 / 자연사 감소 / 동료 충성도 / 암흑 진화 조건

신뢰도 (Trust) — 펫 ↔ 펫
  범위: 0 ~ 100
  영향: 교배 호환성 / 파티 시너지 강화 / 배틀 연계기 발동
  상승: 같은 파티에 오래 있을수록 자연 증가
```

### 성장 등급 (태어날 때 확정, 비공개)
| 등급 | 비율 | 효과 |
|------|------|------|
| 하 | 60% | 렙업당 스탯 낮음 |
| 중 | 30% | 렙업당 스탯 보통 |
| 상 | 10% | 렙업당 스탯 높음 |

### 살아있는 펫 보너스
```
펫이 살아있는 동안: 전체 스탯 +1% 유지
자발적 은퇴 시:     전체 스탯 +0.1% 영구 계승 (다음 펫)
  → 살아있을 때 보너스(+1%)가 더 크지만 영구 계승은 불가
  → 은퇴는 전략적 선택
```

## 8. 진화 시스템

### 진화 단계별 기본 조건 (수치는 조정 가능)
| 진화 | 조건 |
|------|------|
| 유년기 → 성장기 | 레벨 10 + 친화도 20 |
| 성장기 → 완전체 | 레벨 25 + 친화도 40 |
| 완전체 → 궁극체 | 레벨 45 + 친화도 60 |
| 궁극체 → 전설체 | 레벨 70 + 친화도 80 |

### 진화 종류
```
일반 진화:   기본 조건 충족
강화 진화:   기본 조건 + 진화 강화 아이템 (더 강함)
히든 진화:   숨겨진 조건 (캐릭터마다 다름, 비공개)
암흑 진화:   친화도 0% + 암흑 진화 아이템 (필수)
             → 친화도 0% 상태에서 아이템 없이는 진화 불가
```

### 히든 진화 조건 유형 (캐릭터마다 1~3개 조합, 비공개)
```
A. 레벨 요구치 (기본보다 높음)
B. 친화도 극단값 (매우 높거나 0%)
C. 특정 속성 적 N마리 처치
D. 특정 스킬 N회 사용
E. 특정 아이템 보유
F. 위험 상태(체력 10% 이하)에서 전투 승리
G. 특정 컨디션 일정 기간 유지
H. 다른 특정 속성 펫과 높은 신뢰도
I. 특정 사냥터 지역 방문 횟수
J. 실제 시간 조건 (야간, 특정 요일 등)
```

### 히든 진화 스탯 처리
```
히든 진화 = 현재 스탯 유지 + 히든 보너스 추가 (초기화 없음)

단계별 보너스:
  성장기 히든: 현재 스탯 +10% 일괄 상승
  완전체 히든: +15%
  궁극체 히든: +20%
  전설체 히든: +30% + 고유 패시브 스킬 해금
  암흑 진화:   +25% (위험 부담 보상)

→ 빠르게 히든 진화할수록 누적 스탯 유리
```

### 전설 암흑체 (최고 난이도 히든)
```
조건:
  전설체(5단계) 달성
  + 나이 140살 이상 (실시간 70일)
  + 친화도 0%
  + 진화의 저주석 아이템 사용

리스크:
  진화 성공 확률 60%
  실패 확률 40% → 즉사 (소생 아이템으로도 부활 불가)

사전 준비 필수:
  사망률 상쇄를 위한 생명의 부적 +13 강화 3개
  (140살 + 친화도 0% = 케어 보너스 없어 사망률 40%+)
```

## 9. 스킬 시스템

### 스킬 해금 (진화 단계별)
```
유년기: 기본 공격만
성장기: 스킬 1개 해금
완전체: 스킬 2개 해금 (누적 2개)
궁극체: 스킬 3개 해금 (누적 3개)
전설체: 스킬 4개 + 궁극기 1개 해금 (누적 5개)
```

### 스킬 타입
| 타입 | 설명 |
|------|------|
| 액티브 | 직접 발동, 즉각 효과 (공격, 회복 등) |
| 패시브 | 항상 적용, 자동 지속 효과 |
| 버프 | 일정 시간 지속, 아군 강화 or 적 약화 |

```
캐릭터별 스킬 구성 (예시):
  공격형: 액티브 3 + 패시브 1 + 버프 1
  지원형: 액티브 1 + 패시브 2 + 버프 2
  밸런스: 액티브 2 + 패시브 2 + 버프 1
```

### 스킬 강화
```
레벨업 시 스킬 포인트 1개 획득
스킬당 최대 5포인트, 계수 +0.1/포인트
기본 계수: 1.0 → 최대 1.5배
히든 캐릭터: 기본 계수 1.2 (일반 대비 20% 강함)
암흑 진화:   기본 계수 1.3

수동 스킬 발동 시: 효과 +30% (자동 발동 대비)
```

## 10. 컨디션 시스템

### 컨디션 수치 (4종, 0~100)
```
배고픔 / 행복 / 청결 / 체력
감소 속도: -1.5% / 실시간 시간 (오프라인 포함)
```

### 임계값 구간
| 구간 | 범위 | 상태 |
|------|------|------|
| 🟢 안정 | 70~100 | 기본 상태 |
| 🟡 주의 | 30~69 | 경미한 패널티 |
| 🔴 위험 | 0~29 | 심각한 패널티 + 사망률 증가 |

### 패널티 테이블
| 컨디션 | 🟡 주의 (30~69%) | 🔴 위험 (0~29%) |
|--------|-----------------|----------------|
| 배고픔 | 경험치 -20% | 경험치 -50%, 사망률 +2%/day |
| 행복 | 크리티컬 확률 -10% | 크리티컬 -25%, 히든 진화 불가 |
| 청결 | 친화도 성장 -20% | 친화도 -50%, 질병 확률 +10% |
| 체력 | 전투력 -15% | 전투 불가, 사망률 +1%/day |

### 에너지 시스템
```
최대 에너지: 100
회복: 시간 자동 +10/hour / 놀기 활동 +15 / 에너지 물약

활동별 소모:
  사냥 (자동):   -30/회
  사냥 (수동):   -15/회 (절반)
  보물탐사(자동): -25/회
  보물탐사(수동): -13/회 (절반)
  퀘스트:        -10~30/회 (난이도별)
```

## 11. 사망 시스템

### 자연사 확률 (나이 기준)
| 나이 | 기본 사망률/day |
|------|---------------|
| ~49살 | 0% |
| 50살 | 1% |
| 70살 | 5% |
| 100살 | 20% |
| 120살 | 35% |
| 140살+ | 40%+ (최대 50%) |

### 사망률 감소
```
친화도 80+:          -8%/day
모든 컨디션 70%+:    -8%/day
두 조건 모두:        -16%/day

생명의 부적 아이템:  -1% / 개 (강화 시 증가)
  → 케어(-16%) + 아이템 최대 강화(3개 × +13 = -42%) 합산으로 0% 가능
```

### 생명의 부적 강화 시스템
```
기본 효과: -1% / 개
장착 슬롯: 3개 (독립 생명 슬롯)
강화 공식: +n 강화 = -(n+1)% 감소

파괴 확률:
  +1 ~ +3:   파괴 없음 (재료+코인만 소모)
  +4 ~ +6:   20% 파괴
  +7 ~ +9:   40% 파괴
  +10 ~ +12: 60% 파괴
  +13 ~ +15: 75% 파괴
  +16 이상:  85% 파괴

강화 재료 등급:
  하급 강화석: +1~+3 (사냥 드롭)
  중급 강화석: +4~+9 (보물탐사)
  상급 강화석: +10 이상 (고난이도 탐사/보스)
  황금 보호석: 파괴 방지 1회 (극희귀)

강화 비용: 재료비 + 코인 (등급 및 강화 횟수에 따라 증가)
```

### 사냥 중 사망
```
사냥터에서 사망 → 영구 사망 (소생 아이템으로만 부활)
수동 모드:  도망 명령 가능 → 생존 가능
자동 모드:  도망 불가 → 체력 0 시 영구 사망

소생 아이템 (소생의 서):
  코인으로 구매 불가
  사냥/보물탐사/특수퀘스트로만 획득
  사용 시 해당 사냥터에서 부활 (컨디션 30%로)
```

### 자발적 은퇴
```
언제든 자발적 은퇴 가능
살아있을 때 보너스 (+1%) → 은퇴 후 영구 계승 보너스 (+0.1%)로 전환
장기 전략: 오래 살려서 +1% 유지 vs 빠르게 은퇴해서 다음 펫 시작
```

## 13. 전투 시스템

### 데미지 공식
```
데미지 = max(1, 공격력 × 스킬계수 - 방어력) × 상성배율 × 크리티컬배율

방어력 > 공격력 × 스킬계수 시 (방어 초과):
  2%  → 적이 오히려 1 회복
  18% → 0 데미지
  80% → 1 데미지 (최소 보장)
```

### 전투 계수
```
상성배율:     유리 1.3 / 중립 1.0 / 불리 0.77
크리티컬:     기본 10% 확률, 1.5배 데미지
              상성 유리 공격 시: 크리티컬 +15% (최대 25%)
스킬계수:     기본 1.0, 포인트당 +0.1 (최대 1.5)
히든캐릭:     스킬계수 기본 ×1.2
암흑 진화:    스킬계수 기본 ×1.3

수동 스킬 발동: 효과 ×1.3 (타이밍 보너스)
```

### PvE / PvP
```
동일 전투 공식 사용
PvE (사냥): 별도 몬스터 존재 (속성 펫과 무관한 독립 종족)
PvP (온라인 배틀): 실시간 or 자동 방어 (오프라인 시 약화)
```

## 16. 교배 시스템

### 교배 조건
```

히든 캐릭터: 교배 횟수 1~2회

최적 교배 시기: 비공개 (캐릭터마다 다른 나이+레벨 조건)
탄생 시 힌트만 제공 (정확한 시기는 유저가 추적)
```

### 몰빵 시스템
```
보유 교배 횟수를 한 번에 소진 → 히든/특수 확률 집중 상승
예) 5회 권리를 5번에 나눠 쓰기 vs 한 번에 5회 소진
    → 한 번에 소진 시 히든 탄생 확률 대폭 상승
```

### 유전 시스템
```
우성/열성 유전 (비공개)
돌연변이 확률 존재
최적 교배 시기 내 교배 시: 우수한 자질 유전 확률 증가

일반 교배 결과 (같거나 호환 속성):
  → 부모 속성 중 하나 계승

불호환 교배 (성공 시 혼합 속성):
  혼합 × 일반: 95% 일반 / 4% 혼합 유지 / 1% 새 조합

혼합 속성 교배 체인 (옴니렉스 생성 경로):
  증기(물+불) × 바람 → 1% → 물+불+바람
  물+불+바람 × 땅   → 1% → 물+불+바람+땅
  ... 반복 ...       → 전속성 옴니렉스
```

### 온라인 교배 비용
```
교배 신청자 부담:
  1. 교배 기본 비용 (코인, 게임에 소각)
     일반×일반:     100코인
     히든 포함:     500코인
     혼합속성 포함: 1,000코인
  2. 상대 플레이어 지급 (코인 or 아이템, 협의)

자식 귀속: 교배 신청자가 자식 획득
상대방:   코인/아이템 보상 수령
```

## 17. 파티 시스템

### 구성
```
활성 파티: 최대 3마리 (코인으로 4→5슬롯 확장 가능)
비파티 펫: 자유 로밍 (패널티 없음, 독립 사냥/탐사 가능)
```

### 파티 vs 비파티 성장 차이
```
파티 포함 펫:
  경험치 +30%
  스킬포인트 획득 +50%
  주인공 버프 혜택
  파티 시너지 적용

비파티 펫:
  기본 성장 (패널티 없음)
  독립적 자동 사냥/탐사 가능
```

### 파티 포메이션
```
펫 수에 따른 대형:
  1마리: 주인공 우측
  2마리: 주인공 양 옆
  3마리: ▽ 삼각형 (주인공 중심)
  4마리: ◇ 사각형 (슬롯 확장 후)
  5마리: ⬠ 오각형 (최대 확장)

포메이션 보너스 (예시):
  삼각 포메이션: 정면 방어 +5%
  오각 포메이션: 전방위 방어 +8%
```

### 파티 시너지
```
[Tier 1 — 속성 중복]
  같은 속성 2마리: 해당 속성 데미지 +10%
  같은 속성 3마리: +20% + 궁극기 쿨다운 -20%

[Tier 2 — 속성 관계]
  호환 조합 (❤️ 쌍): 공격력 +15%
  불호환 조합 (💔 쌍): 잠재 폭발 +25%, 안정성 -15%

[Tier 3 — 히든 시너지]
  비공개, 유저가 직접 발견
  빛+어둠: 카오스 공명 (특수 스킬 발동)
  신뢰도 높은 조합: 연계기 발동 확률 상승

[신뢰도 연동]
  신뢰도 높은 파티원끼리: 시너지 효과 강화
  신뢰도 낮은 조합: 시너지 미발동 가능
```

## 18. 주인공 (계약자) 시스템

### 기본 설정
```
인간 아바타, 플레이어 그 자체
데스크톱 화면 위를 펫들과 함께 돌아다님
전투 직접 참여 없음 → 버프/아이템 사용 역할
```

### 커스터마이징
```
기본 선택 (게임 시작 시):
  성별:       남/여/중성
  헤어스타일: 4~6종
  의상:       3~4종 기본 제공

해금 요소:
  의상 스킨:  퀘스트/이벤트 보상
  악세서리:   상점 구매 or 탐사 획득
  칭호:       조건 달성 시 해금 ("전설 계약자", "어둠의 테이머" 등)
```

### 계약자 레벨 (Contractor Level)
```
별도 경험치 없음 → 자동 계산
  계약자 레벨 = f(보유 펫 수, 평균 진화단계, 평균 레벨)

레벨에 따라:
  배틀 중 사용 가능 아이템 종류 증가
  버프 사용 횟수 증가
  버프 효과 강도 증가
  파티 전체 패시브 오라 강도 증가
```

### 전투 중 주인공 역할
```
버프 발동:   파티 전체 스탯 일시 상승
아이템 사용: 회복, 디버프 해제, 소생 등
도망 명령:   수동 사냥 시 펫 귀환 명령 가능
             자동 사냥 시: 불가 (영구 사망 위험 유지)
```

## 19. 경제 시스템

### 코인 획득
```
일일 퀘스트:  50~200코인/day
사냥 드롭:    10~500코인/회 (난이도별)
메인 스토리:  500~2,000코인/챕터
보물탐사:     아이템 위주 (코인 적음)
```

### 코인 소모 (주요 코인 소각)
```
가챠 1회:         100코인
가챠 10회:        900코인 (10% 할인)
슬롯 추가:        300 → 600 → 1,200 → 2,400... (2배씩)
교배 기본 비용:   100 ~ 1,000코인 (등급별)
아이템 강화 비용: 재료 등급 × 강화 횟수 (점진 증가)
파티 슬롯 확장:   별도 비용 (4번째, 5번째)
```

### 주요 제한
```
소생의 서: 코인 구매 불가 (사냥/탐사/퀘스트만)
생명의 부적: 상점 구매 가능, 강화는 재료 필요
진화의 저주석: 코인 구매 불가 (탐사 전용)
```

## 24. 확장 계획

### Phase 1 — MVP (현재 목표)
```
일반 8속성 기본 라인 (40 스프라이트)
히든 2속성 기본 라인 (10 스프라이트)
핵심 시스템: 육성/진화/사냥/교배/파티
단일 플레이어 위주
```

### Phase 2 — 온라인 확장
```
온라인 교배/교환/배틀 서버
랭킹/친구 시스템
클라우드 세이브
```

### Phase 3 — 콘텐츠 확장
```
혼합 속성 캐릭터 라인 추가
히든 진화 다양화
추가 속성 (땅, 번개, 얼음, 독 외 확장)
이벤트 시스템 / 시즌 콘텐츠
스토리 추가 챕터
```

### 확장 가능 설계 원칙
```
속성: 데이터 파일로 정의, 하드코딩 금지
캐릭터: 독립 데이터 파일 (추가 시 파일 하나)
호환성 차트: 데이터 기반 (A↔B 대칭 자동 보장)
히든 진화 조건: 조건 유형 enum으로 조합식 정의
```
