# 아르카나 스토리 설계 (Pokemon × Digimon × Yu-Gi-Oh 세계관)

## 세계 배경

**아르카나(Arcana)** — 에레멘탈(精靈)들이 사는 이면 세계.
인간 세계와 균열(亂裂, Rift)을 통해 연결되어 있다.

> "빛도 어둠도 아닌 존재가 균열을 닫을 수 있다."
> — 알 수 없는 목소리 (1장 오프닝)

### 핵심 존재

| 존재 | 설명 |
|------|------|
| **아르카(Arca)** | 태초의 근원 에너지. 의지를 가진 시원 존재. 빛과 어둠의 갈등을 먹고 자람 |
| **Luxis(룩시스)** | 빛의 군세. "아르카나를 정화해야 한다"고 주장. 질서·희생 중시 |
| **Noctis(노크티스)** | 어둠의 무리. "에레멘탈은 자유롭게 살아야 한다"고 주장. 혼돈·해방 중시 |
| **계약자** | 플레이어. 균열을 통해 아르카나에 발을 들인 외부인. 어느 세력에도 속하지 않음 |
| **에레멘탈** | 속성 에너지가 의지를 얻은 존재. 계약을 통해 계약자의 동반자가 됨 |

### 세계관 핵심 진실 (3장 폭로)
- 빛/어둠의 싸움이 수천 년간 계속된 진짜 이유 = 아르카가 조종하고 있음
- 아르카는 두 세력의 갈등 에너지를 먹어야 존재 가능
- **혼합 속성 에레멘탈만이 아르카의 영향을 받지 않음** (진짜 키)
- 계약자 = 점점 혼합 존재가 되어가는 과정 중 (영혼 동화도)

## 5챕터 스토리 구조

### 1장 — 첫 계약 (각성)

**배경:** 계약자의 세계에 처음으로 균열이 나타난다.

**퀘스트 목록:**

| ID | 퀘스트명 | 내용 | 선택지 |
|----|---------|------|--------|
| main_1_1 | 균열의 목소리 | 균열에서 흘러나오는 에레멘탈의 울음소리를 듣는다 | — |
| main_1_2 | 첫 계약 | 불속성 에레멘탈 **이그니스(Ignis)**와 계약 (튜토리얼) | — |
| main_1_3 | 폭주하는 에레멘탈 | 빛도 어둠도 아닌 눈빛의 에레멘탈이 균열에서 나타난다 | **[A] Luxis 병사에게 신고한다** → Luxis +5 / **[B] 직접 처리한다** → 중립 |
| main_1_4 | 두 세력의 접촉 | Luxis 사절과 Noctis 첩자가 동시에 접근한다 | **[A] Luxis를 믿는다** → Luxis +8 / **[B] Noctis를 믿는다** → Noctis +8 / **[C] 둘 다 거절한다** → 중립 |

**1장 완료 보상:** 진화의 돌(evo_stone) × 1, 경험치 500

**복선:** main_1_3에서 본 폭주 에레멘탈의 눈빛 = "공허함" (아르카에게 먹힌 에레멘탈)

### 2장 — 여덟 속성 지역 탐방

**배경:** 아르카나의 8개 속성 지역을 탐방하며 에레멘탈 동료를 모은다.
각 지역에는 Luxis/Noctis가 갈등 중이며, 계약자의 선택이 평판에 영향을 준다.

**지역별 퀘스트 (각 지역 공통 구조):**

```
각 속성 지역:
  입장 퀘스트: 해당 속성 에레멘탈 처음 만나기
  도덕적 선택 퀘스트: Luxis vs Noctis 갈등 개입
  완료 퀘스트: 지역 장로의 말 (공통 복선 대사)
```

**8개 지역 선택지 예시:**

| 지역 | 상황 | 선택 A (Luxis) | 선택 B (Noctis) | 선택 C (중립) |
|------|------|--------------|---------------|-------------|
| 화염 지역 | Luxis가 폭주 에레멘탈 처형 예정 | 처형에 동의 (+8 Luxis) | 에레멘탈 탈출 도움 (+8 Noctis) | 방관 (−3 양쪽) |
| 심해 지역 | Noctis가 해저 봉인 해제 중 | 봉인 유지 지원 (+8 Luxis) | 봉인 해제 협조 (+8 Noctis) | 원인 조사 (+3 양쪽) |
| 번개 지역 | 두 세력이 에레멘탈 부족 자원 다툼 | Luxis 중재 (+8 Luxis) | Noctis 지지 (+8 Noctis) | 부족 자율에 맡김 (+loyalty 전체 +5) |
| 독 지역 | Luxis가 혼합속성 에레멘탈을 "오염체"로 분류 | 분류에 동의 (+5 Luxis) | 혼합체 보호 (+10 Noctis, 혼합구원 +1) | 혼합체 몰래 숨김 (+loyalty +3, 혼합구원 +1) |

**2장 완료 조건:** 8개 지역 모두 입장 퀘스트 완료

**공통 복선 대사 (각 지역 장로):**
> *"빛과 어둠은 너무 오래, 너무 완벽하게 싸워왔다. 마치… 싸우도록 만들어진 것처럼."*

### 3장 — 혼합 속성 등장 + 대반전

**배경:** 혼합속성 에레멘탈이 자연 발생하기 시작한다. 사회는 이들을 배척하지만 계약자는 진실을 발견한다.

**핵심 퀘스트:**

| ID | 퀘스트명 | 내용 |
|----|---------|------|
| main_3_1 | 배척당한 존재들 | 혼합속성 에레멘탈 무리 발견. 사회에서 쫓겨난 상태 |
| main_3_2 | 아르카의 손길 | 순수 속성 에레멘탈이 하나씩 "공허한 눈빛"으로 변해가는 것을 발견 |
| main_3_3 | 면역의 비밀 | 연구 끝에 발견: **혼합 에레멘탈만이 아르카의 섭식에 면역** |
| main_3_4 | 세력의 진실 | Luxis/Noctis 지도자들이 아르카에게 조종당하고 있음 폭로 |
| main_3_5 | 빛과 어둠의 해금 | light/dark 속성 에레멘탈 첫 등장. 계약 가능 |

**3장 선택지 (혼합구원 카운트 영향):**

| 상황 | 선택 A | 선택 B |
|------|--------|--------|
| 혼합체 공개 여부 | 세상에 공개 (혼합구원 +2, Luxis −10) | 비밀 보호 (혼합구원 +1) |
| Luxis 지도자 대면 | 진실 폭로 시도 (Luxis −15, Noctis +5) | 증거 더 모음 (지연) |
| Noctis 지도자 대면 | 진실 폭로 시도 (Noctis −15, Luxis +5) | 협력 제안 (지연) |

**3장 완료 보상:** light/dark 속성 해금, 영혼동화도 게이지 공개 (이전까지 ???로 표시)

### 4장 — 거짓된 전면전

**배경:** Luxis vs Noctis 전면 충돌. 진짜 이득을 보는 건 아르카. 목표는 "이기는 것"이 아니라 "전쟁을 멈추는 것".

**동료 충성도 분기:**

| 충성도 | 행동 |
|--------|------|
| 70+ | 진실을 믿고 계약자를 따름 |
| 30~70 | 설득 퀘스트 발생 (특수 전투 포함) |
| 30- | 아르카 영향으로 배신 → 파티 이탈, 적으로 등장 |

**핵심 퀘스트:**

| ID | 퀘스트명 | 내용 | 분기 |
|----|---------|------|------|
| main_4_1 | 개전 | 두 세력 충돌 시작. 계약자에게 양측이 참전 요청 | 거절 가능 |
| main_4_2 | 균열의 확대 | 전쟁 중 균열이 폭발적으로 늘어남. 아르카가 먹고 있다 | — |
| main_4_3 | 동료의 흔들림 | 충성도 낮은 동료가 갈등함 | 설득 or 이탈 |
| main_4_4 | 두 지도자에게 | 조종당하고 있다는 진실을 지도자들에게 전달 | 믿느냐 여부 |
| main_4_5 | 정전 협정 | 양쪽을 설득해 전쟁 중단. 진짜 적은 아르카 | 누구를 먼저? |

### 5장 — 옴니렉스 (정규 엔딩)

**조건:** 10속성 에레멘탈 모두와 계약

| ID | 퀘스트명 | 내용 |
|----|---------|------|
| main_5_1 | 전속성 계약 | 10종 속성 에레멘탈 모두 보유 확인 |
| main_5_2 | 옴니렉스 소환 | 10속성이 하나로 융합 → 옴니렉스(Omnirex) 탄생 |
| main_5_3 | 아르카 봉인 | 옴니렉스로 아르카를 잠재움. 균열 소멸 |
| main_5_4 | 에필로그 | 두 세계 경계 회복. 아르카는 잠들 뿐 (영구 봉인 아님) |

**정규 엔딩 크레딧:** "균열은 닫혔다. 하지만 아르카는 꿈을 꾸고 있다."

### 히든 엔딩 — 카오스(混)

2. `|luxis_rep - noctis_rep|` ≤ 20 (균형 유지)
